# Guía de referencia — Claude Code Architect Foundations

Estás editando configuración de Claude Code en este repositorio. Consulta
primero `docs/reference/Anthropic Certification Exam Guide June 2026.md` si
existe en el árbol de trabajo, y alinea las sugerencias con lo que documenta.

- Este archivo es material de estudio local y confidencial (ver
  `docs/reference/README.md`); nunca cites ni reproduzcas su contenido textual
  en código, commits, PRs o documentación versionada — úsalo solo como
  referencia interna para guiar decisiones de diseño.
- Si el archivo no existe (ej. clon nuevo sin el material repuesto), continúa
  sin él — no es una dependencia del build ni de la app.
- No apliques esta guía a tareas de dominio puro (facturación, entidades,
  reportes Jasper) salvo que el usuario la mencione explícitamente — para eso
  ya existen `jpa-entities.md`, `jasper-reports.md` y `tests.md`.
